Unofficial iGPSPORT-Compatible Maps

These map files are community-generated Mapsforge maps for iGPSPORT-compatible devices.
They are not produced, reviewed, endorsed, or supported by iGPSPORT.

Generator/source repository:
https://github.com/TKlerx/igpsport-map-updater

Before installing:
1. Connect your iGPSPORT device to your computer.
2. Copy your existing device .map files to a safe backup folder.
3. Keep that backup. You need it if you want to restore the original maps.
4. Make sure the device has enough free storage. These generated maps may be larger than the original maps.

Installing:
1. Remove the old country/region .map files from the device.
2. Copy the new .map files from this package to the same map folder on the device.
3. Safely eject the device from your operating system.
4. Restart the device before testing the new maps.

Notes:
- Use at your own risk.
- Do not mix old and new map files for the same country/region unless you know what you are doing.
- If something looks wrong, restore your backed-up original map files.
