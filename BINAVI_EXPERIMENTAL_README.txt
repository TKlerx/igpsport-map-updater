Experimental BiNavi-Compatible Maps

These files are an experimental community-generated BiNavi map package.
They are not produced, reviewed, endorsed, or supported by iGPSPORT.

What is updated:
- The main parseable country .map file in Maps/ is replaced with a generated Mapsforge map.

What is preserved from the official BiNavi package:
- Router/*.rtd files
- Non-standard Maps/*.map files, including contour/overview files

Important limitations:
- Router/*.rtd files are not regenerated.
- Contour/elevation maps are not regenerated.
- The generated main map may not contain the same POI, landuse, rail, or contour data as the official BiNavi maps.

Before installing:
1. Connect your iGPSPORT/BiNavi device to your computer.
2. Copy the existing map package from the device to a safe backup folder.
3. Keep that backup. You need it if you want to restore the official maps.
4. Make sure the device has enough free storage.

Installing:
1. Remove the old matching country map package from the device.
2. Copy the Maps/ and Router/ folders from this package to the same location on the device.
3. Safely eject the device from your operating system.
4. Restart the device before testing the new maps.

Use at your own risk. If anything looks wrong, restore your backed-up official package.
